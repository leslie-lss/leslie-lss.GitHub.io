## jekyII && hexo && hugo

### jekyII

github pages默认选用jekyII进行博客的渲染，jekyII是基于Ruby的静态网页生成系统，需要安装Ruby。

### hexo

* hexo选用nodejs进行页面渲染，需安装nodejs；  
* 可使用markdown进行博客文本的编辑；  
* 利用sublime text3 && 相关插件进行markdown的编辑和预览；  

### hugo

hugo采用Go语言进行博客的编写，需要学习Go语言。
